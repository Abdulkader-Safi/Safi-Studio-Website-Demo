LaralCN-UI

A shadcn-style component system for Laravel. Not a runtime library you install and couple to. You pull pure Blade and Tailwind v4 components into your own source tree and fully own them, edit them, ship them, no version chasing.

Open the docs View on GitHub
install

$ php artisan ui:init

→ tokens written to app.css

$ php artisan ui:add button card dialog

✓ 3 components copied to your repo

$ php artisan ui:list

25 available · 3 installed

Why it's different

  • ✓Copy-and-own. The components live in your repo, not in vendor/. Edit anything, no fork needed.
  • ✓Two ways in: an Artisan CLI for Laravel, or one-click copy from the docs for plain Blade and Statamic.
  • ✓One registry as the single source of truth, so both paths output identical files.
  • ✓Tailwind v4 CSS-first theming, with tailwind-merge so your classes always win.
  • ✓Alpine.js only where interactivity is unavoidable. No JavaScript framework tax.
